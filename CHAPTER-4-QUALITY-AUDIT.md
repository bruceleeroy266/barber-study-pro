## 4. Quiz Audit

### Count: 35 questions (comprehensive quiz) + 4 questions (mini quiz in HTML)

### Question Distribution
| Topic Area | Count | Assessment |
|------------|-------|------------|
| General safety/shop practices | ~15 | Too many; dilutes infection control focus |
| OSHA/EPA/SDS | ~4 | Adequate |
| Disinfection procedures | ~5 | Good |
| Hand washing / PPE | ~4 | Good |
| Exposure incidents | ~3 | Adequate |
| Pathogens / disease | ~2 | **Severely underrepresented** |
| Electrical / fire safety | ~2 | Off-topic for infection control |

### Critical Issues
- 🚨 **All 35 answers are "B"** — This is a massive design flaw. Students will game the system.
- 🚨 **No explanations** for wrong answers — Students don't learn from mistakes
- 🚨 **No question types other than multiple choice** — No true/false, no matching, no scenario-based
- 🚨 **Too many generic safety questions** — "What shoes should you wear?" is not infection control
- 🚨 **No questions on bacterial shapes** — Major test topic missing
- 🚨 **No questions on specific pathogens** — Hep B, HIV, fungi barely tested
- 🚨 **No questions on disinfectant types** — Quats, phenolics, tuberculocidal not tested
- 🚨 **No scenario-based questions** — Real-world application not assessed
- 🚨 **No difficulty progression** — All questions same complexity
- 🚨 **Mini quiz (4 questions) and comprehensive quiz (35 questions) have no overlap** — Inconsistent coverage

### Recommendations
- **Randomize answer positions** — Never have all answers in the same position
- **Add explanations** for every wrong answer
- **Add scenario-based questions** — "What do you do if...?"
- **Add matching questions** — Match pathogen to disease
- **Add true/false questions** — Good for misconceptions
- **Rebalance topics** — More pathogen questions, fewer generic safety questions
- **Add difficulty levels** — Easy/medium/hard
- **Add a "hardest questions" section** for test prep
