# The Great Ark - Minecraft Server Setup

## BEFORE YOU START

You need to download the Minecraft server software FIRST.

## Step 1: Download Server Software

1. Open your web browser
2. Go to: https://www.minecraft.net/en-us/download/server/bedrock
3. Click the **Windows** download button
4. Save the ZIP file
5. Extract (unzip) the file INTO THIS FOLDER

## Step 2: Start the Server

1. Double-click **START.bat**
2. A black window will open
3. Wait for it to say "Server started"
4. Leave this window OPEN while playing

## Step 3: Connect in Minecraft

1. Open Minecraft
2. Click Play
3. Click Servers
4. Click Add Server
5. Enter:
   - Name: The Great Ark
   - Address: localhost
   - Port: 19132
6. Click Save
7. Click on The Great Ark to join!

## For Kids to Connect

Give them your computer's IP address:
1. Open Command Prompt
2. Type: ipconfig
3. Find "IPv4 Address"
4. They use that number instead of "localhost"

## Server Settings

- Name: The Great Ark
- Game Mode: Survival
- Difficulty: Normal
- Max Players: 20

## Troubleshooting

**Black window closes immediately?**
- You didn't download the server software yet (Step 1)

**Can't connect in Minecraft?**
- Make sure the server window is still open
- Try restarting Minecraft
- Check Windows Firewall (allow bedrock_server.exe)

**Server not showing up?**
- Make sure you're using "localhost" as the address
- Try the Friends tab instead of Servers
